+ for，while条件语句后要加":"，不需要使用"{}"

## 变量
python中变量不用指定类型直接使用=进行初始化。

	i = 1
	m, n = 1, 2
可以使用var = None的方法声明变量，不进行初始化赋值。

## list
列表使用"[]"，可使用":"取范围[:]

	l = []
	l = [1, 2, 3]	

## dict
字典类型，使用"{}"

	tel = {'jack': 4098, 'sape': 4139}
	tel['guido'] = 4127
